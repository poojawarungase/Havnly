# Havnly

Havnly is a full-stack web application inspired by Airbnb, enabling users to discover, book, and list unique accommodations worldwide. Built with React, Node.js, Express, and MongoDB, it offers a seamless experience for travelers and hosts alike.


# Features

User authentication (sign up, login, logout)

Browse and search listings with filters

View detailed property pages with images and amenities

Book accommodations with date selection and availability checks

Hosts can add, edit, and remove their property listings

Secure JWT-based authentication with cookies

Responsive design for mobile and desktop

# Tech Stack

Layer = Technology

Frontend = React, Vite, Tailwind CSS

Backend = Node.js, Express
